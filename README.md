<h3 align="center">An Android Movie Database Application</h3>

## About
Netplix allows you to look through movie database and search through it. 

## What was used
- Kotlin
- MVVM Design pattern
- themoviedb API
- Jetpack Compose
- Dagger-Hilt
- Coroutine
